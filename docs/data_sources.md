# Data Sources

MM-NeuroOnco aggregates multiple publicly available medical imaging datasets.

This repository does **NOT** redistribute raw medical images.

All users must download the original datasets directly from their official sources and comply with their respective licenses and usage agreements.

## Compliance Statement

MM-NeuroOnco aggregates metadata and derived annotations from publicly available datasets for academic research purposes.

We do **not** claim ownership of the underlying medical images.

All rights to the original images remain with their respective dataset providers.

Users are responsible for:

- Reviewing and complying with the license terms of each original dataset.
- Ensuring proper attribution when required.
- Using the data strictly within permitted research boundaries.

If any dataset provider has concerns regarding inclusion in this benchmark, please contact the repository maintainers.
