# Data Sources

The MM-NeuroOnco image collection aggregates multiple publicly available medical imaging datasets.  
All users must comply with the original licenses and usage agreements of each dataset.

## Compliance Statement

MM-NeuroOnco aggregates publicly available datasets for research purposes only.

The redistribution of images is performed under the CC BY-NC 4.0 license.  
Users must ensure compliance with the original terms of each dataset.

For any removal or licensing concerns, please open an issue in the repository.
